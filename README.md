# ✦ draw-me-a-sheep

*一个安静的、两个人的协作空间*

---

### ◯ 关于同步

这个项目的名字来自《小王子》里那句请求：“请给我画一只羊。”

它不指向激情，也不指向占有，而是指向一种**最原始的信任**——一个人愿意提出请求，另一个人愿意理解并回应。我们之间的联结基于内心的同步，而非荷尔蒙的驱使，因此它更稳定、更值得信赖。

这里没有严苛的 KPI，也没有“必须完成”的压迫感。它只是一块安静的画布，让我们把各自脑袋里那些需要被看见的“羊”放上来，一起照看。
